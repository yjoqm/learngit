说明和示例
Check Vdnagen Ver 王杰 无 返回一串字符，为VDNAGen -v的输出
${result} = Check Vdnagen Ver

=>

${result} = "VDNAGen 3.4.4.0"
Generator Far 王杰 cmd，dir_far，media_file VDNAGen的标准输出，格式为xml

可以作为VDNAGen中的任何Generator使用，指定目录结构，具体目录结构如下：
dir_far目录：CI_SCRIPT_ROOT/data/VDNAGen_data/source_file/
media_file目录：CI_SCRIPT_ROOT/data/VDNAGen_data/generator_far/
CI_SCRIPT_ROOT：该目录在搭建自动化测试环境时，需要在系统的环境变量中配置

${result} = Generator Far VDNAGen c.flv.far c.flv

=>

${result} = <?xml version="1.0" encoding="utf-8"?>
<receipt>
<ErrorCode>0</ErrorCode>
<ErrorMsg>Success</ErrorMsg>
</receipt>
Generator Far Dna 王杰 media_file，dir_far VDNAGen的标准输出，格式为xml

作为VDNAGen命令中的将dna文件生成far文件使用，命令自带-TDNA参数
media_file和dir_far参数会自带CI_SCRIPT_ROOT目录，该目录在搭建自动化测试环境时，需要在系统的环境变量中配置

${result} = Generator Far data/c.flv.dna far/c.flv.far

=>

${result} = <?xml version="1.0" encoding="utf-8"?>
<receipt>
<ErrorCode>0</ErrorCode>
<ErrorMsg>Success</ErrorMsg>
</receipt>
Generator Far For Batch 王杰 无 返回格式为列表，列表里包含一个或多个VDNAGen的标准输出，格式为xml

批量执行VDNAGen中的将media文件生成far使用，指定目录结果，具体目录结果如下：
原文件目录：CI_SCRIPT_ROOT/data/VDNAGen_data/source_file/
生成far文件的目录：CI_SCRIPT_ROOT + '/data/VDNAGen_data/generator_far
每个far文件的命名为：原文件名 + .far

${result} = Check Vdnagen Ver

=>

${result} =

[" <?xml version="1.0" encoding="utf-8"?>
<receipt>
<ErrorCode>0</ErrorCode>
<ErrorMsg>Success</ErrorMsg>
</receipt>",
" <?xml version="1.0" encoding="utf-8"?>
<receipt>
<ErrorCode>0</ErrorCode>
<ErrorMsg>Success</ErrorMsg>
</receipt>"]

Parse Dna Status Result 王杰 data 以字典的格式输出 把dna_status工具解析出来的输出进行一定程度上的解析，然后按字典的格式返回
目前该返回值包含的key有：LENGTH，Total length is，version is，VIDEO
所有的values均为str格式
Parse Ffmpeg Result 王杰 data 以字典的格式输出 把ffmpeg工具解析出来的输出进行一定程度上的解析，然后按字典的格式返回
目前该返回值包含的key有：Duration（在Duration以转换成秒，且返回格式为str）
