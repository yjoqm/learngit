Introduction

本库是用于Excel读写。支持后缀为xls的excel文件和后缀为xlsx的excel 2007文件，目前该库只支持文件读取。
如果某单元格的值是一个公式或者宏，该库读取到的是这个单元格最终的值而非公式本身

本库的关键字中涉及的序号全部从1开始而非0开始，比如要选择表格的第一行和第三列使用关键字Get Cell Value | 1 | 3，而非Get Cell Value | 0 | 2，其他类同

使用方法：
读取表格时需要先使用关键字Open Excel，然后使用关键字Select Sheet选择一张表格，之后才能使用其他的关键字读取表格中的内容，否则将出错
